## Audience: Game Masters

## Description - 
This compiler aims to help with the daunting task that is preparing for a TTRPG campaign by helping Game Masters organize their notes and encounters in an easily understood format for Virtual Tabletop Softwares as well as note taking apps. Furthermore, it allows for the conversion of prior notes into these new formats.

For example, it will convert a .ds file into a structure output file such as a .json or .md, which are readily used in numerous Virtual Tabletop softwares like FoundryVTT as well as note taking apps like Obsidian.
